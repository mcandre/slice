# slice: text sampler

slice discards random lines from STDIN.

# EXAMPLES

```console
$ cd examples

$ slice <romeo-and-juliet.txt
  Escalus, Prince of Verona.
  Friar John, Franciscan.
  Three Musicians.
  An Officer.
  Nurse to Juliet.
    In fair Verona, where we lay our scene,
                                                         [Exit.]
```
